NaijaWedCalculator is a simple web-based Nigerian wedding budget estimator.

It helps couples and planners estimate the cost of a wedding in Nigeria based on guest count, wedding type, and city. All calculations run locally in the browser. No accounts, no tracking, and no personal data collection.

What this app does

Estimates total wedding cost instantly
Supports common Nigerian cities and regions
Adjusts pricing based on wedding type (Traditional, White, or Both)
Shows a clear budget breakdown (venue, catering, attire, decoration, media, entertainment, miscellaneous)
Allows users to edit individual costs manually
Lets users add custom expenses
Includes light and dark theme support
Works fully offline once loaded

Why this exists

Wedding planning in Nigeria is expensive and often stressful. Costs are rarely transparent. This tool gives couples a realistic starting point so they can plan smarter, negotiate better, and avoid surprises.
